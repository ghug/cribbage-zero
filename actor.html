♣ Cribbage Zero — Actor

Self-play on this device in the background and upload sample shards to the data bus for the learner. Runs as a foreground service with a wake-lock, so it keeps going with the screen off. ← tools

Data bus: — set the bus URL and your worker token in ⚙ Settings.

The on-device actor runs only inside the Cribbage Zero app (it needs the native engine + a background service — a browser can't self-play). Install the APK from Releases, open it, and come back to this page. To watch the live run, see the Training monitor; to contribute from a PC, run cz_pc — see the tools hub.

The actor pulls the latest net from GitHub, self-plays, and uploads sample shards to the bus, refreshing the net whenever the learner advances it. The worker token only lets this device append self-play. Version · public domain (The Unlicense).
