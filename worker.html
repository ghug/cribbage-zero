This device is an actor: it self-plays against the learner's net (read from the public GitHub net branch) and uploads every sample it produces (in chunks) to the data bus. The screen is kept awake while running. The token only lets this device append self-play — it can't read or change anyone else's data; the net is read anonymously from GitHub.
